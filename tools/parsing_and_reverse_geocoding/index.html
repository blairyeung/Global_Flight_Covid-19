ISO 3166:2 - awesome source

This little widget is built to take locality data that is distributed with debian and convert it into JSON objects for inputting into MongoDB (or any other storage system), there a few steps to take to ensure that everything works tickety-boo:

  1. Get the most up-to-date iso-codes from debian (they seem to be the most actively developed set of data)
  2. Compare the data in /iso-codes/iso_3166_2/iso_3166_2.xml and make sure it lines up with the iso newsletters
  3. All good? Copy the entire /iso-codes directory into the directory containing this html file
  4. Click the below button and check your console
  5. As we're geocoding the locations through GMaps (with a rate-limit), the results will appear on the map above :-)
